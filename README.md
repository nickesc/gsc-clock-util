# GSC Clock Password Generator

A web-based tool to generate passwords for resetting the real-time clock (RTC) in Gen II Pokémon games: *Pokémon Gold, Silver & Crystal*.

## Usage

1. Navigate to [nickesc.github.io/gsc-clock-util](https://nickesc.github.io/gsc-clock-util/) in a web browser
2. Enter your trainer information:
   - Trainer Name
   - Trainer ID
   - Money
   - Game Region
3. Click "Generate Password"
4. Use the generated password in the game's clock reset menu
